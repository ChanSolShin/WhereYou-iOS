# WhereYou(웨어유)

<aside>

### 앱 이름

- WhereYou(웨어유)

---

### 앱 소개

- 모임 약속에 속해있는 멤버의 위치 공유를 통한 약속 시간 준수 앱

---

### 앱 목적

- 지정된 시간에만 실시간으로 위치를 공유하여 서로의 도착 여부와 위치를 확인할 수 있도록 도와주는 앱 

---

### 사용 기술

- **Language:** Swift
- **Framework:** SwiftUI
- **Architecture:** MVVM
- **Backend:** Firebase (Firestore, Authentication, Functions, RealtimeDB)
- **MAP**: ****NAVER Maps API
- **Asynchronous:** GCD, Swift async/await
- **Version Control:** Git & GitHub
- **Collaboration**: Notion

---

## 배운점 및 성과

- Firebase를 활용한 클라이언트-서버 데이터 처리 경험 확대
- MVVM 아키텍처 및 비동기 데이터 처리 능력 향상
- 앱스토어 출시를 통한 실무적 배포, 유지보수 경험 확보
- 실시간 데이터를 안정적으로 다루는 구조적 접근법 습득
- UIUX 디자이너의 필요성을 인지 → V0 도입의 계기
